# Pharmacy Database System (PL/SQL)

## Overview

The Pharmacy Database System is a database management project developed using Oracle SQL and PL/SQL. The system is designed to manage pharmacy operations, including customer records, medicine inventory, sales transactions, and reporting functionalities.

This project demonstrates database design, relational modeling, SQL querying, and PL/SQL programming concepts.

## Database Features

* Customer management
* Medicine inventory management
* Sales tracking
* Report generation
* Data validation and integrity
* Relational database design

## Technologies Used

* Oracle Database
* SQL
* PL/SQL
* Oracle SQL Developer

## Database Structure

The system consists of several related entities, including:

* Users
* Customers
* Medicines
* Sales
* Reports

Relationships between tables are implemented using primary keys and foreign keys to ensure data consistency and integrity.

## Implemented Concepts

### SQL

* SELECT statements
* JOIN operations
* Aggregate functions
* Sorting and filtering
* Data manipulation (INSERT, UPDATE, DELETE)

### PL/SQL

* Procedures
* Functions
* Variables and control structures
* Exception handling
* Database transactions

## Learning Objectives

This project was developed to strengthen knowledge of:

* Relational database design
* SQL querying
* PL/SQL programming
* Database normalization
* Data management systems

## How to Run

1. Open Oracle SQL Developer.
2. Create a new database connection.
3. Execute the provided SQL scripts to create the tables.
4. Run the PL/SQL scripts to create procedures and functions.
5. Insert sample data and test the database operations.
